# Web App para Control de Mercaderia 

 Este es un proyecto de control de mercaderias averiadas hechas para una empresa llamada MULTICAR Repuestos

 ## Descripción

 Pagina Web creada para practicar HTML, CSS y algo de JS.
 Incluye una estructura basica que permite guardar, editar, borrar, buscar y seleccionar productos guardados

 ## Tecnologías

 - HTML5
 - CSS3
 - JavaScript

## Notas

Se utilizaron herramientas de documentación y asistencia de IA para apoyo en partes específicas del desarrollo.
Fue desarrollado durante una etapa de aprendizaje, por lo que la estructura del codigo puede contener inconsistencias en el estilo o la estructura.
Cambios posteriores podrian mejorarlo al no ser la version final.
